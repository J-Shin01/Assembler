## Intro
- 이 프로젝트는 “The Elements of Computing Systems – Noam Nisan “ 책에서 설정한 컴퓨터 아키텍처를 바탕으로
- Nand 소자부터 어셈블러까지 개발한 프로젝트이다.  
- 책에서는 Hack 컴퓨터의 아키텍처와 Hack 어셈블리의 문법을 제공한다. 이를 참고하여 Hack 어셈블러를 개발하였다.


## 개발 순서
- HDL로 논리 게이트 개발
- 논리 게이트로 가산기, 증분기 개발
- ALU 개발
- 레지스터, RAM 개발
- 컴퓨터 개발(일명 ‘Hack 컴퓨터’)
- Hack 어셈블러 개발
